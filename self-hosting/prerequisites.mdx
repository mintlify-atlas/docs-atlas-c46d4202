Before you deploy skyhan.cloud, make sure you have the required software installed and an optional analytics account if you want usage tracking.

## Required software

<CardGroup cols={2}>
  <Card title="Node.js" icon="node" href="https://nodejs.org/en/download/">
    JavaScript runtime environment required to build and run the Next.js application.
  </Card>

  <Card title="pnpm" icon="box" href="https://pnpm.io/installation">
    Fast, disk-efficient package manager used by this project. Install it with `npm install -g pnpm`.
  </Card>

  <Card title="Git" icon="git-branch" href="https://git-scm.com/downloads">
    Version control system used to clone the repository to your machine.
  </Card>

  <Card title="Rybbit (optional)" icon="chart-bar" href="https://www.rybbit.io">
    Analytics platform. Create an account to get a site ID and instance URL. Skip this if you don't need analytics.
  </Card>
</CardGroup>
