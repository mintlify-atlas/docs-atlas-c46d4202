## Prerequisites

Before you begin, make sure you have the following installed:

- [Node.js](https://nodejs.org/en/download/) (v18 or later)
- [pnpm](https://pnpm.io/installation) (v9 or later)
- [Git](https://git-scm.com/downloads)

## Run locally

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/skyhancloud/skyhan.cloud.git
    ```
  </Step>

  <Step title="Navigate to the project directory">
    ```bash
    cd skyhan.cloud
    ```
  </Step>

  <Step title="Install dependencies">
    ```bash
    pnpm install
    ```
  </Step>

  <Step title="Configure environment variables">
    Copy the example environment file and fill in your values:

    ```bash
    cp .env.example .env
    ```

    Open `.env` and set the analytics variables (both are optional):

    ```env
    RYBBIT_ANALYTICS_SITE_ID=
    RYBBIT_ANALYTICS_URL=
    ```

    <Note>
      Analytics are entirely optional. If you leave these values empty, no analytics scripts are loaded. The development server works fully without them.
    </Note>
  </Step>

  <Step title="Start the development server">
    ```bash
    pnpm dev
    ```

    The site is now running at [http://localhost:3000](http://localhost:3000). The dev server uses Next.js Turbopack for fast refresh.
  </Step>
</Steps>

## Available scripts

| Script | Command | Description |
|---|---|---|
| Development | `pnpm dev` | Starts the dev server with Turbopack |
| Build | `pnpm build` | Creates an optimised production build |
| Start | `pnpm start` | Serves the production build |
| Lint | `pnpm lint` | Runs ESLint across the codebase |
