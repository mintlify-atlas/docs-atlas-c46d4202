The homepage at `app/page.tsx` is divided into two sections that make up the portfolio: **Projects** and **Works**. Both sections are rendered as standalone React components.

## Analytics events

Both components fire [Rybbit](https://rybbit.io) custom events when a visitor clicks a link.

```typescript
// Fired when a project link is clicked (components/projects.tsx)
window.rybbit.event('project_link_click', {
  project_name: item.name,
  project_link: item.link
});

// Fired when a work link is clicked (components/works.tsx)
window.rybbit.event('work_link_click', {
  work_name: item.name,
  work_link: item.link
});
```

<Note>
  Events are only fired if the `rybbit` object is present on `window`. When analytics are disabled or blocked, clicks still navigate normally.
</Note>
