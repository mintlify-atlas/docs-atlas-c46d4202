## Shared utilities

### `cn` utility

All components that conditionally apply Tailwind classes use the `cn` utility from `utils/cn.ts`. It combines [`clsx`](https://github.com/lukeed/clsx) for conditional class logic with [`tailwind-merge`](https://github.com/dcastil/tailwind-merge) to resolve conflicting Tailwind utilities.

```typescript
import cn from '@/utils/cn';

// Merges classes safely, resolving Tailwind conflicts
cn('px-2 py-1', isActive && 'bg-green-500', 'px-4');
// => 'py-1 px-4' (px-2 is overridden by px-4)
```

### `next-view-transitions` `<Link>`

Every component that renders a navigation link imports `Link` from [`next-view-transitions`](https://github.com/shuding/next-view-transitions) rather than `next/link` directly. This enables the View Transitions API for smooth page animations across the site.

```typescript
import { Link } from 'next-view-transitions';
```
