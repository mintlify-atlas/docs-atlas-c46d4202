## What is skyhan.cloud?

skyhan.cloud is a personal website and blog built with **Next.js 15** and **Tailwind CSS**. It serves as a central hub for Gökhan's projects, professional work history, blog posts, and live Discord presence status.

The site is open-source under the GNU General Public License v3.0, meaning you can freely read the code, learn from it, or self-host your own version.

## Tech stack

| Layer | Technology |
|---|---|
| Framework | Next.js 15 (App Router) |
| Language | TypeScript |
| Styling | Tailwind CSS |
| MDX | `@next/mdx` + `@mdx-js/react` |
| State | Zustand |
| Validation | Zod |
| Analytics | Rybbit |
| Presence | lantern.rest |
| Fonts | Geist Sans, Faustina |

## Key features

- **Portfolio** — Showcase projects with names, descriptions, and links
- **Work history** — Chronological list of professional engagements
- **Blog** — MDX-powered articles with syntax-highlighted code blocks
- **Discord status** — Real-time online/offline status and Spotify listening info via [lantern.rest](https://lantern.rest)
- **Dark / light mode** — Warm color palette that adapts to the system preference
- **View transitions** — Smooth animated page transitions via `next-view-transitions`
- **Analytics** — Privacy-friendly analytics via Rybbit (optional)
